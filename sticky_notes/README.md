## 🌟 Overview

The **Sticky Notes Application** is a beginner-friendly Django project that demonstrates core web development concepts including:

- **Model-View-Template (MVT)** architecture
- **Object-Relational Mapping (ORM)** with Django models
- **CRUD operations** (Create, Read, Update, Delete)
- **URL routing** and pattern configuration
- **Template inheritance** and dynamic content rendering
- **Form handling** with Django's ModelForm
- **Static file management** (CSS)
- **Database migrations**
- **Django Admin** customisation


---

## ✨ Features

- ✅ **Create new sticky notes** with title, content, and colour
- ✅ **View a list of all notes** in a responsive grid layout
- ✅ **View individual note details** on a dedicated page
- ✅ **Update existing notes** with pre-filled forms
- ✅ **Delete notes** with confirmation prompts
- ✅ **Choose from 5 colours** (yellow, green, pink, blue, purple)
- ✅ **Timestamps** for creation and last update
- ✅ **Django Admin** interface for backend management
- ✅ **Responsive design** that works on mobile and desktop
- ✅ **Clean, modern UI** with custom CSS styling
